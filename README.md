# Crashlog::AuthHMAC

A ruby gem for authenticating requests using HMAC

## Installation

Add this line to your application's Gemfile:

    gem 'crashlog-auth-hmac'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install crashlog-auth-hmac
